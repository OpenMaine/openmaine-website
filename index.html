OpenMaine is a former Code for America Brigade.

We volunteer to help design, develop, and improve civic systems in Maine. Whether you are a professional coder or a community member with an idea, everyone interested in making Maine better is welcome!

About


We are a former brigade of the Code for America network. Code for America uses the principles and practices of the digital age to improve how government serves the American public, and how the public improves government.

You can find our Code of Conduct here. As a new organization we are still drafting important documents, including a mission statement, values, and project guidelines. You can find these documents on our Github account and we welcome feedback on them.
